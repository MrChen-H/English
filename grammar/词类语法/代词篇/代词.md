# 代词

代词在英语中往往表达指代某物或某人,其类型分为:

- [*人称代词*](#人称代词)

- [*物主代词*](#物主代词)

- [*反身代词*](#反身代词)

- [*指示代词*](#指示代词)

- [*不定代词*](#不定代词)

代词分单数和复数形式,部分代词还分各种人称(详情见下文)

# 人称代词

<br>

><font size = 5>定义</font>

- 人称代词即指代人称的代词(你,我,他,她),其一般分为三种人称还区分主格和宾格与单复数(见下表)

||第一人称|第二人称|第三人称 (男)|第三人称 (女)|第三人称 (其他)|
|:-:|:-:|:-:|:-:|:-:|:-:|
|**主格**|I|You|He|She|It|
|**主格复数**|We|You|They|They|They|
|――|
|**宾格**|Me|Yuo|Him|Her|It|
|**宾格复数**|Us|You|Them|Them|Them|

# 形物代词

<br>

><font size = 5>定义</font>

- 形容词性物主代词(形物代)是物主代词的一种,置于名词前,起修饰作用，表示某人的...   
  其各种人称和形式如下

||第一人称|第二人称|第三人称 (男)|第三人称 (女)|第三人称 (其他)|
|:-:|:-:|:-:|:-:|:-:|:-:|
|**单数**|My (我的)|Your (你的)|His (他的)|Her (她的)|its (它的)|
|**复数**|Our (我们的)|Your (你们的)|Their (他们)|Their (他们的)|Their (它们的)|

# 名物代词

<br>

><font size = 5>定义</font>

- 名词性物主代词是在物体已经指出的情况下用来代替已知物体的物主代词，后面不需要加上已知名词   
  其形式如下

||第一人称|第二人称|第三人称 (男)|第三人称 (女)|第三人称 (其他)|
|:-:|:-:|:-:|:-:|:-:|:-:|
|**单数**|Mine (我的)|Yours (你的)|His (他的)|Hers (她的)|its (它的)|
|**复数**|Ours (我们的)|Yours (你们的)|Theirs (他们)|Theirs (他们的)|Theirs (它们的)|

# 物主代词相关语法
 
<br>

- 形物代词必须和名词连用

        eg: I will go to see my aunt on Sunday

        译文: 我将会在周日去看望 我的 阿姨.

<br>

- 名物代词需单独使用

        eg: Your classroom is very big , but ours is rather samll.

        译文: 你的教室很大,但是 我们的(指前文提到的教室) 非常小

<br>

- Ps: 形物代词 + 名词 == 名物代词

        eg: These are my books  ==  These books are mine(My books).

        译文: 这是我的书


        解释: 形物代词其实就是把名物代词省略的名词写出来了

# 指示代词

><font size =5>定义与用法</font>

<br>

- 指示代词一般为 it that noe 用于指代前文出现的事物

        eg: I have lost my umbrella,and I'm looking for it.

        翻译: 我的伞丢了,我正在找它(指代伞)
        
        解释: it 的指代非常精准,必须是同名同物的指代,且 it 可以指代可数名词也可以指代不可数名词。


        I have lost my umbrella,and I must buy one.

        翻译: 我的伞丢了,我必须再买一把(指代前文的伞但不是同一把)

        解释: one 一般为泛指,即同名不同物体,需注意的是 one 只能指代可数名词单数


        The umbrella you bought is cheaper that I bought.

        翻译: 你买的伞比的那把更便宜(That 指代前文的 The umbrella)

        解释: That 特指前文已经出现的 The + 名词 ,that 可以指代可数名词也可以指代不可数名词(复数用those)

# 不定代词

<br>

><font size =5>定义与用法</font>

- 不定代词通常是泛指,如some,every等, 详情如下

        单数不定代词:
	    some,any,no,none,each,every,noe,either,neither,the other,another,much,little,a little,all

	    复数不定代词：
	    many,few,a few ,noes,both,others,the others
	
	    复合不定代词:
	    some/any/no/every + noe/body/thing
	
	    All    -> 指三者或者三者以上的可数或不可数都
	    Both -> 指二者都

	    All/Both of the 名词		所有都


	    No / None
	    No+可数名词/不可数名词 ,表示没有
	    None :可以单独使用，表示"没有一个(人或物)"

	    every  /  each	每个
	    every: 形容词+名词(不能单独使用)
	    each:  可以做形容词和every一样，也可以单独使用(侧重于复数中的个体)

	    either / enither 
	    either: 两者中的一个，either ....or 不是...就是...(用于否定句句末表也)
	    neither: 两者都不	  neither...nor... 既不也不	Neither he nor you are a student.  他和你都不是学生
        (注意be动词使用了 are 是因为neither遵循就近原则)
